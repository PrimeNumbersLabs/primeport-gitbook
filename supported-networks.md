# Supported Networks

PrimePort is a multichain NFT marketplace. You can browse, trade, and interact with NFTs on the following networks.

***

## Base (Chain ID: 8453)

[Base](https://base.org) is an Ethereum Layer 2 built by Coinbase. It offers low gas fees, fast transaction finality, and full EVM compatibility.

**What's on Base:**
- **PRFI ONFTs** — The yield-bearing NFT collection from Prime Numbers Labs
- NFT staking via the PRFI Staker contract
- Trading via PrimePort and OpenSea

| Detail           | Value                                                                                |
| ---------------- | ------------------------------------------------------------------------------------ |
| **Chain ID**     | 8453                                                                                 |
| **Native token** | ETH                                                                                  |
| **RPC URL**      | `https://mainnet.base.org`                                                           |
| **Block Explorer** | [basescan.org](https://basescan.org)                                               |
| **Bridge**       | [bridge.base.org](https://bridge.base.org)                                           |

***

## XDC Network (Chain ID: 50)

The [XDC Network](https://xdc.org) is an enterprise-grade, EVM-compatible blockchain known for near-zero gas fees and fast block times.

**What's on XDC:**
- **XDC Staking NFTs** — Active staking collection (stake psXDC inside your NFT for XDC rewards)
- Legacy PRNT Staking NFTs (historical — no longer actively used)
- Third-party NFT collections
- Full PrimePort marketplace features: minting, drops, auctions, offers

| Detail           | Value                                                                                |
| ---------------- | ------------------------------------------------------------------------------------ |
| **Chain ID**     | 50                                                                                   |
| **Native token** | XDC                                                                                  |
| **RPC URL**      | `https://erpc.xinfin.network`                                                        |
| **Block Explorer** | [xdcscan.com](https://xdcscan.com)                                    |

> **Address format:** XDC addresses use the `xdc` prefix instead of `0x`. When sending funds from an exchange, replace `0x` with `xdc`.

***

## Switching Networks

PrimePort detects your wallet's active network automatically. To switch between XDC and Base:

1. Open your wallet (MetaMask, Rabby, etc.)
2. Select the network you want to use
3. PrimePort will update to show collections and features for that network

Some features (drops, minting, contract deployment) are currently available only on the XDC Network. Trading and staking for PRFI ONFTs are available on Base.

***

## Future Networks

PrimePort is built to be chain-agnostic. Additional EVM-compatible networks may be added in the future as the ecosystem grows.
